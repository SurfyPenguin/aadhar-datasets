## Project Overview
Aadhar enrollment data visualization and prediction system with:
- Streamlit dashboard for data visualization
- PyTorch ML model for enrollment prediction
- Monthly and yearly enrollment analysis
- District-wise and age-group breakdowns

## Features
- Interactive state and district selection
- Monthly enrollment trends visualization
- ML-based enrollment prediction by month/year
- Age group analysis (0-5, 5-17, 18+)
- District-level prediction insights
- Growth rate calculations and trend analysis

## Technology Stack
- Python 3.12+
- Streamlit
- PyTorch
- Pandas
- Plotly
- UV package manager
